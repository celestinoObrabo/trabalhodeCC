Desenvolvimento

O SAOD faz uso extensivo das classes de utilitário Bootstrap 4 para evitar o inchaço do CSS e o baixo desempenho da página.

Classes CSS personalizadas pelos desenvolvedores Celestino e Matheus Gurgel (alunos do IFRN) caíram de boa serventia para proporcionar ao usuário a melhor experiência possível com o uso desse sistema.
